ul --> (Nieuporządkowana lista)


list-style: none; --> (Jest regułą CSS stosowaną do usuwania domyślnych stylów punktacji lub numeracji elementów listy w HTML )


margin: 5px; -->(Odstęp poza borderem 5px w każdą stronę )


padding: 0px; -->(Zeruje odstęp w środku )


ul a:link, -->(Selektor dla linków )


ul a:visited -->(Selektor dla odwiedzonych stron )


color: #005500; --> (Ustawia kolor tekstu )


text-decoration: none;--> (To reguła CSS, która eliminuje domyślną dekorację dla hiperłączy )


ul a:hover --> ( Oznacza styl, który jest stosowany do hiperłączy 'a' wewnątrz listy nieuporządkowanej 'ul' w momencie, gdy kursor myszy najedzie na ten link )


text-decoration: underline; -->(To reguła CSS, która dodaje podkreślenie do tekstu )


display: block; -->(Element będzie wyświetlony w bloku (odstęp z góry i z dołu)


width: 150px; -->(Ustawia szerokość elementu na 150 pikseli)


list-style: none; -->( Po zastosowaniu tej reguły punkty lub liczby w liście nie będą wyświetlane, co jest przydatne, gdy chcesz stworzyć niestandardowe układy listy bez domyślnego formatowania.)


margin: 0px; -->(Ustawia margines elementu na 0 pikseli)


border: 2px outset #cccccc; -->(Stosuje obramowanie o grubości 2 pikseli do elementu. Obramowanie ma styl "outset", co oznacza, że brzegi będą wyglądać jak wyglądające na wypukłe. Kolor obramowania jest ustawiony na #cccccc, co jest odcieniem szarości)


padding: 5px;-->(Dodaje wewnętrzny odstęp o szerokości 5 pikseli do elementu )


border-style: inset; -->(ustawia styl obramowania na "inset", co powoduje, że brzegi obramowania wyglądają jakby były wypukłe do środka elementu )
